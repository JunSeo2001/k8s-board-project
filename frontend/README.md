# CN Board Frontend

CN Board 게시판 프론트엔드 애플리케이션입니다.

## 기술 스택

- React 18
- Vite
- React Router DOM
- Axios
- React Query (TanStack Query)

## 설치 및 실행

### 1. 의존성 설치

```bash
npm install
```

### 2. 개발 서버 실행

```bash
npm run dev
```

애플리케이션이 http://localhost:3000 에서 실행됩니다.

### 3. 빌드

```bash
npm run build
```

## API 서버 설정

백엔드 서버가 `http://localhost:8080`에서 실행되어야 합니다.

Vite 개발 서버의 프록시 설정을 통해 `/api` 요청이 자동으로 백엔드로 전달됩니다.

## 주요 기능

- 게시글 목록 조회
- 게시글 상세 조회
- 게시글 작성
- 게시글 수정
- 게시글 삭제
